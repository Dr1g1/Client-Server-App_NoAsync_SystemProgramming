# Task 13:  
Create a web server that counts words within a file. Count only words that start with a capital letter and have more than 5 letters. All requests to the server are sent via a browser using the GET method. The filename is provided as a parameter in the request. The server receives the request, searches the root folder and all its subfolders for the requested file, and performs the word count. If the requested file does not exist, return an error to the user. Example server request: http://localhost:5050/file.txt
